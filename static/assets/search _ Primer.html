Usage

You can use this icon in React, Ruby, or Jekyll. Here are code examples for each:

React

<SearchIcon size={16} />

ViewComponent

<%= render(Primer::Beta::Octicon.new(:search, size: :small)) %>

Jekyll

{% octicon search height:16 %}
